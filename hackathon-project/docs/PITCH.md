# PredictPulse — 2-Minute Pitch Script

## [0:00 - 0:15] Hook

"Prediction markets are a billion-dollar industry — but here's the dirty secret: not all predictions are created equal. Some have an 85% track record of accuracy. Others? Barely better than a coin flip. And right now, there's no tool to tell the difference."

## [0:15 - 0:30] Problem

"Platforms like Metaculus, Polymarket, and Kalshi show you a probability — say, 62% — but they don't tell you whether that number is actually reliable. Is it based on 5 forecasters or 500? Is it in a domain where crowd wisdom historically works, or one where it fails? Traders, researchers, and decision-makers are flying blind."

## [0:30 - 0:45] Solution

"PredictPulse changes that. We built an end-to-end data science pipeline on Zerve that trains a machine learning model on 500+ resolved predictions. It learns which signals — participation density, confidence patterns, economic context — actually predict accuracy. Then it scores any active prediction with a reliability rating and a plain-English explanation powered by Claude AI."

## [0:45 - 1:15] Demo

"Let me show you. I enter a question: 'Will global temperature exceed 1.5°C before 2030?' The community says 62%. PredictPulse analyzes the metadata — 245 forecasters, high engagement ratio, strong participation density — and returns an 82% reliability score with a 'High' rating. Claude explains exactly why: the combination of expert participation and sustained engagement in the Science category historically correlates with accuracy. But it also flags caveats — external shocks aren't captured."

## [1:15 - 1:35] Technical Depth & Zerve Showcase

"Under the hood, we engineered 20+ features, trained and compared three model architectures with 5-fold cross-validation, and deployed the best performer as a live API — all within Zerve's platform. Data collection, feature engineering, model training, visualization, AI analysis, and deployment — six connected blocks, one seamless workflow."

## [1:35 - 1:50] Impact & Vision

"PredictPulse could improve prediction-based decision making by 15-25%. Phase 2 adds multi-platform support. Phase 3 is a browser extension that shows reliability scores right alongside prediction market odds. We're turning crowd noise into signal."

## [1:50 - 2:00] Close

"PredictPulse — because the question isn't 'What does the crowd predict?' It's 'Should you believe them?' Thank you."
